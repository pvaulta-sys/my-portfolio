# Ogunleye Amos Ayodeji — Systems Builder

Personal headquarters for **Ogunleye Amos Ayodeji** — founder, software developer, financial analyst, and systems builder creating infrastructure products across **finance**, **logistics**, and **security**.

## Products showcased

- **Vaulta** — Enterprise treasury operating system
- **Courier platform** — Real-time logistics & shipment tracking
- **AEGIS** — Security operations platform (dashboard + CLI)

## Stack

Next.js 16 · React 19 · Tailwind CSS 4 · Framer Motion · Lucide React

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Test on your phone (same Wi‑Fi)

The dev server listens on all interfaces (`0.0.0.0`). On your Mac, find your LAN IP (e.g. **192.168.1.6**), then on your phone open:

`http://192.168.1.6:3000`

Use your machine’s real IP — not `localhost`. Restart `npm run dev` after pulling these changes so `allowedDevOrigins` and the hostname bind take effect.

## Configuration

Edit `lib/site.ts` (social URLs, Cal link), `lib/projects.ts`, and `lib/domains.ts` for copy and product data. Thinking essays live in `lib/thinking.ts` + `components/thinking/`.
